# Personal Portfolio Website

## Project Overview

This is a responsive personal portfolio website built using HTML, CSS, and Bootstrap. It is designed to showcase my skills, projects, and contact information in a clean and professional layout.

## Features

- Fully responsive design (mobile, tablet, desktop)
- Fixed navigation bar with smooth scrolling
- Home, Skills, Projects, and Contact sections
- Modern UI using Bootstrap 5
- Interactive project cards with hover effects
- Clean and simple layout

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
